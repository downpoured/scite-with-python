
### SciTE-with-Python

A fork of the SciTE editor that adds Python scriptability

Also adds dozens of plugins and the following features: default keyboard bindings can be changed, menu items can be removed, current search term is shared between SciTE instances. 

### Windows

* install [Python 2.7](https://www.python.org/downloads/windows/), use the Windows x86 MSI installer

* download [scite_with_python_0_7_1_win32.zip](https://github.com/downpoured/scite-with-python/releases/download/v0.7.1/scite_with_python_0_7_1_win32.zip)

* uncompress everything in the .zip

* open SciTE.exe (no "install" needed)

### Linux

    (install prereqs)
    sudo apt-get install libgtk2.0-dev
    sudo apt-get install python2.7-dev
    
    (build scintilla)
    cd ~/Downloads
    wget https://github.com/downpoured/scite-with-python/archive/v0.7.1.tar.gz
    tar -xzf v0.7.1.tar.gz
    cd scite-with-python-0.7.1/src/scite/scintilla/gtk
    make
    
    (build scite)
    cd ../../scite/gtk
    make
    sudo make install

    (run scite)
    SciTE_with_python

### Building from source on Windows

download [Microsoft Visual C++ Compiler for Python 2.7](https://www.microsoft.com/en-us/download/confirmation.aspx?id=44266)

(you must use this compiler version for C runtime compatibility)

save VCForPython27.msi to a directory, open an admin cmd prompt, and run

`msiexec /i VCForPython27.msi ALLUSERS=1`

then, open a cmd prompt that can run git, go to the desired directory, and run

`git clone https://github.com/downpoured/scite-with-python.git`

then, open a cmd prompt, and run the following

`cd "C:\Program Files (x86)\Common Files\Microsoft\Visual C++ for Python\9.0"`

if there is no such directory / you are running a 32bit version of windows,

`cd "C:\Program Files\Common Files\Microsoft\Visual C++ for Python\9.0"`

then run

`vcvarsall.bat`

`cd "c:\(path to scite-with-python)\src\scite\scintilla\win32"`

`nmake -f scintilla.mak`

`cd "c:\(path to scite-with-python)\src\scite\scite\win32"`

`nmake -f scite.mak`

unzip `src/scite/scite/python/libs.zip` into `src/scite/scite/python/libs`

unzip `src/scite/scite/bin/pythonbinaries.zip` into `src/scite/scite/bin`

unzip `src/scite/scite/bin/tools_internal/wincommondialog/wincommondialog.zip` into `tools_internal/wincommondialog/`

run `"c:\(path to scite-with-python)\src\scite\scite\bin\SciTE.exe"`
