  • zip 形式 (1265K) 一般的に Windows 用です。
  • tgz 形式 (1100K) 一般的に Linux とその互換 OS 用です。
  • zip format (1265K) commonly used on Windows
  • tgz format (1100K) commonly used on Linux and compatible operating systems

Windows

Windows 用実行可能ファイルWindows Executables

フルダウンロード (620K) では SciTE の実行可能ファイルと必要な DLL, 設定ファイル、解説書が含まれています。ファイルをダウンロードしたら zip を展開して SciTE.EXE を実行してください。実行のために必要なファイルは SciTE.EXE, SciLexer.DLL, SciTEGlobal.properties です。これらは同一のパスにある単一のディレクトリ内にあるのが一番よい位置です。

A full download (620K) includes the SciTE executable, any required DLLs, configuration files and documentation. After downloading the file, unzip it, and run SciTE.EXE. The files required to run SciTE are SciTE.EXE, SciLexer.DLL, and SciTEGlobal.properties and these are best located in one directory on the path.

Sc1と呼ばれる単一実行可能ファイル (370K) は DLL ファイルや属性ファイル相当分を実行ファイルにリンクしているのでそれらを必要としません。属性ファイルは依然作ることができます。Sc1.exe は UPX コンプレッサで圧縮されているのでダウンロード時間を短くすることができます。使用前に展開する必要はありません。

A single file executable called Sc1 (370K) does not need any DLL or properties files as these are linked into the executable. You may still create properties files if you wish. Sc1.exe has been compressed with the UPX compressor so that it is a fast download. It does not need to be decompressed to be used.

A distribution based on SciTE CVS snapshots which includes C++, Perl, PHP, Java and nncron APIs. Syntax highlighting for Perl, PHP and C++ differs from the original distribution and there are other cosmetic changes. Contributed by Serge Baranov.

GTK+ / Linux

Intel 互換プロセッサ用 Linux 実行可能ファイルLinux executable for Intel compatible processors

このバイナリリリースは GTK+ 2.4 以上が必要です。Fedora Core 2 で検査されました。一ヶ月以上古い Linux ディストリビューションを使っているときは、あなたの手元の GTK+ で使うためにおそらく GTK+ 2.4 をインストールするか SciTE を再構築する必要があるでしょう。

This binary release requires GTK+ 2.4 or later and was tested on Fedora Core 2. If you are using a Linux distribution more than a month old you probably need to install GTK+ 2.4 or rebuild SciTE to use your existing version of GTK+.

A フルダウンロード (640K) は SciTE の実行可能ファイルと設定ファイル、解説書を同梱しています。ダウンロードした後 gunzip と untar にかけて展開し SciTE を実行してください。SciTE の実行に必要なファイルは SciTE （という名前のファイル） と 40 個の属性ファイルです。前者は実行パス(私は /usr/local/bin に入れています)、後者は /usr/share/scite に置いてください。

A full download (640K) includes the SciTE executable, configuration files and documentation. After downloading the file, gunzip and untar it, and run SciTE. The files required to run SciTE are SciTE which is best located on the path (I put it in /usr/local/bin) and 40 properties files which should be located in the /usr/share/scite directory.
