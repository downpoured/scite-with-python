ソースコードSource Code

ソースコードパッケージは Scintilla の全てのソースコードを含んでいますが、実行可能バイナリは含まれていません。

The source code package contains all of the source code for Scintilla but no binary executable code and is available in

  • zip 形式 (590K) 一般的に Windows 用です。
  • tgz 形式 (500K) 一般的に Linux とその互換 OS 用です。
  • zip format (590K) commonly used on Windows
  • tgz format (500K) commonly used on Linux and compatible operating systems

Windows 用も Linux 用もビルドための基礎情報については readme ファイルにあります。

Instructions for building on both Windows and Linux are included in the readme file.

Windows 用実行可能ファイルWindows Executables

Scintilla の DLL だけをダウンロードすることはできませんが、SciTE 実行可能ファイル完全ダウンロードに SciLexer.DLL として収められています。

There is no download available containing only the Scintilla DLL. However, it is included in the SciTE executable full download as SciLexer.DLL.
